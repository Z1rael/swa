== Introduction and Goals

=== Requirements Overview

_What is *PhotoHive*?_

The main purpose of PhotoHive is creating a social media plattform with a focus on image sharing

*_Main features_*

Account Management (Login, Register)
Image Upload
Liking & Commenting Images
Filtering & Editing Images (ThirdParty integration Pixlr)
Premium Version with extra features (Cloud storage and more editing)

=== Quality Goals

[options="header",cols="1,1,2,4"]
|===
|Priority|Key|Quality|Motivation
| 1 | | Usability | High usability is a must, if we want to be competitive against the existing platform giants.
| 2 | | Scalability | Because of our size it is important, that we can scale to serve a massive international audience.
| 3 | | Auditability | Because of our international operation, auditability is important to be able to respect the various laws.
| 4 | | Reliability | Reliability is important, because nothing is more annoying to an end user than a page that does not work.
| 5 | | Portability | To reach the largest audience it is important, that the application can run on most devices.
|===

=== Stakeholders

[options="header",cols="1,2,2"]
|===
|Role/Name|Contact|Expectations

| User
| End users (students / public users)
| Wants to share, edit and interact with images, wants the application to be fast and usable on different platforms

| Admin
| System administrators / moderation team 
| Wants to have the necessary tools to moderate the platform, curate a collection of highlighted images and remove problematic content

| Developer
| development team 
| Wants to deliver modular features fast and safely, maintain a clean and extensible architecture

| Influencer
| Content creators / social media users
| Wants to build an audience, attract other users, grow a community and potentially monetize content

| Legal Team
| University compliance / legal advisory (GDPR / copyright compliance)
| Wants the application to adhere to international and local legal requirements to avoid lawsuits, especially regarding personal data and licensing (copyright)
|===
